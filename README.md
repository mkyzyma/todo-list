# Todo list feathers.js/mobx example

Test task for hiring

## Start
```
yarn start
```

## Test
Go to 
```
http://localhost:3030
```

## Routes
-*** Get Todo list
```
GET http://localhost:3030/todos
```
- Get Todo by id
```
GET http://localhost:3030/todos/id
```
- Add Todo

```
POST http://localhost:3030/todos
```
data:
```
{ "title": "Task title" }
```
